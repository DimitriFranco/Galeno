# Parsifal to Zotero Integration

### 📑 Sobre o Projeto

Este software oferece uma solução eficiente para a integração entre a plataforma *Parsifal* e o gerenciador de referências *Zotero*. O objetivo principal é automatizar e simplificar a extração de listas de artigos, permitindo que pesquisadores e acadêmicos organizem suas revisões sistemáticas com muito mais facilidade e agilidade.

### 🚀 Funcionalidades

  * *Integração Direta:* Conexão otimizada entre Parsifal e Zotero.
  * *Exportação Facilitada:* Obtenha listas de artigos prontas para uso sem processos manuais exaustivos.
  * *Interface Amigável:* Desenvolvido para focar na produtividade do usuário.
